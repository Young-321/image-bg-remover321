## 📋 接下来的步骤

### 第一步: 创建D1数据库

1. 访问 Cloudflare Dashboard: https://dash.cloudflare.com/
2. 进入 **Workers &amp; Pages** → **D1**
3. 点击 **Create database**
4. 数据库名称: `bg-remover-users`
5. 点击 **Create**

### 第三步: 获取数据库ID

创建数据库后，在数据库详情页面找到 **Database ID**，复制它。

### 第四步: 更新Worker配置

1. 进入 **Workers &amp; Pages** → **bg-remover-worker** → **Settings**
2. 点击 **Variables and secrets** → **Add binding**
3. 选择 **D1 Database**
4. **Variable name**: `DB`
5. **Database**: 选择 `bg-remover-users`
6. 点击 **Save**

### 第五步: 重新部署Worker

在本地执行：

```bash
cd /root/.openclaw/workspace/bg-remover/worker
./deploy-worker.sh
```

### 第六步: 部署前端

```bash
cd /root/.openclaw/workspace/bg-remover
npm run build
npm run deploy
```
