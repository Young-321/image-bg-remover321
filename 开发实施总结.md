## 🎯 核心功能说明

### 1. 用户分层策略

#### 未登录用户
- **配额**: 3次免费，永久有效
- **存储**: localStorage
- **用完引导**: 显示登录提示

#### 登录用户
- **配额**: 每月5次免费，每月1日重置
- **存储**: 数据库
- **用完引导**: 显示充值提示

#### 付费用户
- **配额**: 登录用户免费配额 + 付费套餐配额
- **叠加使用**: 优先使用免费，再使用付费

---

### 2. 配额管理 (useQuota Hook)

**核心功能**:
```typescript
// 主要API
const { 
  quotaState,      // 配额状态
  isLoading,       // 加载状态
  deductQuota,     // 扣除配额
  refreshQuota     // 刷新配额
} = useQuota(userId);
```

**配额状态结构**:
```typescript
interface QuotaState {
  userType: 'guest' | 'logged_in' | 'paid';
  
  // 未登录用户配额
  guest: {
    total: 3;
    used: number;
    remaining: number;
  };
  
  // 登录用户月度配额
  loggedIn: {
    year: number;
    month: number;
    total: 5;
    used: number;
    remaining: number;
  } | null;
  
  // 付费用户配额
  paid: {
    total: number;
    used: number;
    remaining: number;
    validUntil: number | null;
  } | null;
}
```

## 📱 用户体验流程

### 未登录用户流程

```
访问首页
    ↓
显示3次免费配额
    ↓
用户上传图片
    ↓
配额检查
    ↓
配额充足 → 处理图片，扣除1次配额
配额不足 → 显示登录提示，引导登录
```

### 登录用户流程

```
登录成功
    ↓
初始化5次月度配额
    ↓
用户上传图片
    ↓
配额检查
    ↓
配额充足 → 处理图片，扣除1次配额
配额不足 → 显示充值提示，引导充值
```
