# AI 背景移除网站 - 技术方案总结

## 🎯 一句话总结
为普通用户打造的零成本、隐私友好的在线图片背景移除工具，采用 **Next.js 静态前端 + Cloudflare Workers 边缘后端 + Remove.bg API** 架构。

## 🏗️ 核心架构

### 前端 (静态部署)
- **技术**：Next.js 14 + TypeScript + Tailwind CSS
- **部署**：Cloudflare Pages（全球 CDN）
- **特点**：完全静态，无需服务器，加载速度快

### 后端 (边缘计算)
- **技术**：Cloudflare Workers
- **位置**：全球 300+ 边缘节点
- **特点**：内存处理，无持久化存储，零服务器运维

### AI 服务
- **服务商**：Remove.bg API
- **免费额度**：50张/月
- **收费**：超出后 $0.29/张
- **质量**：专业级抠图效果

## ✨ 三大优势

### 1. 成本优势
- **Cloudflare Workers**：10万次/天免费
- **Cloudflare Pages**：无限请求免费
- **Remove.bg**：50张/月免费
- **总成本**：个人使用基本零成本

### 2. 性能优势
- **边缘计算**：用户就近处理，延迟 < 100ms
- **无服务器**：自动扩展，无单点故障
- **纯内存**：图片不落盘，处理完即释放

### 3. 隐私优势
- **无存储**：不保存用户图片
- **无追踪**：不记录用户信息
- **端到端**：全链路 HTTPS 加密

## 🚀 处理流程

```
用户上传图片 (前端)
    ↓
FormData 包装 (前端)
    ↓
发送到 Worker (Cloudflare 边缘)
    ↓
调用 Remove.bg API (AI 处理)
    ↓
返回 PNG 图片 (透明背景)
    ↓
用户下载 (前端)
```

## ⚡ 快速启动

### 1. 环境准备
```bash
# 克隆项目
git clone <repo>
cd bg-remover

# 安装依赖
pnpm install
```

### 2. API 配置
1. 访问 https://www.remove.bg/api
2. 注册免费账号
3. 获取 API 密钥

### 3. 本地运行
```bash
# 前端开发
pnpm dev          # http://localhost:3000

# Worker 开发
pnpm worker:dev   # http://localhost:8787
```

### 4. 生产部署
```bash
# 部署 Worker
pnpm wrangler secret put REMOVE_BG_API_KEY
pnpm wrangler deploy

# 前端部署（GitHub → Cloudflare Pages）
pnpm build  # 输出到 out/ 目录
```
